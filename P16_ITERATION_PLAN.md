# P16: 移动端适配 + 引导优化 + 版本更新

## 目标
优化移动端体验、简化引导流程、添加版本更新日志

## 功能清单

| ID | 功能 | 描述 | 优先级 |
|----|------|------|--------|
| P16-1 | 移动端适配 | 响应式布局优化、侧边栏折叠、汉堡菜单 | P0 |
| P16-2 | SetupWizard简化 | 减少步骤、增加快速开始选项 | P0 |
| P16-3 | What's New弹窗 | 版本更新日志，首次访问显示 | P1 |
| P16-4 | 连接引导优化 | 数据源配置体验优化、示例配置 | P1 |

## 验收标准
- [ ] 移动端(<768px)正常显示，无布局错乱
- [ ] 侧边栏可折叠，不遮挡主要内容
- [ ] SetupWizard从6步减少到3步（欢迎→快速配置→完成）
- [ ] What's New弹窗在版本变更后首次访问显示
- [ ] vue-tsc --noEmit 无错误

## 技术方案

### P16-1: 移动端适配
1. AppSidebar.vue: 添加移动端汉堡菜单按钮
2. 侧边栏在移动端默认收起，点击按钮展开
3. 卡片间距和内边距在移动端调整
4. 搜索栏在移动端吸顶

### P16-2: SetupWizard简化
1. 合并WelcomeStep + TourStep → 新的欢迎页（品牌+快速开始按钮）
2. 合并ConfigStep + ImportStep → 连接配置页（支持自动检测）
3. DoneStep单独一页
4. 新增"快速开始"按钮：使用预设示例数据，无需手动配置

### P16-3: What's New弹窗
1. 新增WhatsNewModal组件
2. 后端/api/changelog端点返回版本更新内容
3. 前端localStorage记录已读版本号
4. 版本变更时自动弹出

### P16-4: 连接引导优化
1. ConfigStep添加"检测现有配置"按钮
2. 添加示例配置（preset）供一键导入
3. 显示连接状态指示器（成功/失败）
4. 添加"跳过配置"选项，使用默认内存数据
