## 功能清单

### P18-1: 快速操作栏 (Quick Access Bar)

**问题**: 用户经常需要在不同视图间切换（收藏、搜索、图表），需要多次点击

**解决方案**: 在顶栏添加可配置的快速操作按钮，一键访问常用视图

**实现方案**:
- 在 SearchBar 下方添加快速操作栏
- 默认显示: 全部记忆、收藏、最近7天、图表
- 可通过右键菜单自定义显示哪些按钮
- 存储到 localStorage

**验收标准**:
- [ ] 快速操作栏显示在搜索栏下方
- [ ] 点击按钮快速切换视图
- [ ] 可自定义显示/隐藏哪些按钮
- [ ] 配置持久化到 localStorage

---

### P18-2: 记忆对比视图 (Memory Comparison View)

**问题**: 用户需要对比两条记忆的差异时，没有好的可视化方式

**解决方案**: 并排展示两条记忆，高亮显示差异部分

**实现方案**:
- 新增 `frontend/src/views/CompareView.vue`
- 从记忆卡片右键菜单"对比"触发
- 左右两栏展示，使用 diff 算法高亮差异
- 支持 JSON 和 Markdown 格式

**验收标准**:
- [ ] 选中两条记忆后可以打开对比视图
- [ ] 差异部分用黄色高亮显示
- [ ] 支持 JSON 和 Markdown 格式
- [ ] 关闭按钮正常返回

---

### P18-3: 键盘快捷键自定义 (Keyboard Shortcut Customization)

**问题**: 高级用户希望自定义键盘快捷键以匹配自己的工作流

**解决方案**: 在设置页面添加快捷键自定义功能

**实现方案**:
- 扩展 `frontend/src/composables/useKeyboard.ts`
- 添加 localStorage 配置存储
- 新增 `frontend/src/components/ShortcutSettings.vue`
- 冲突检测：不允许重复绑定

**验收标准**:
- [ ] 设置页面显示所有可自定义的快捷键
- [ ] 可以修改快捷键绑定
- [ ] 修改后的快捷键立即生效
- [ ] 冲突检测提示

---

### P18-4: 智能内存建议 (Smart Memory Suggestions)

**问题**: 用户不知道该创建什么类型的记忆，系统缺乏主动建议

**解决方案**: 基于现有记忆模式，智能推荐可以创建的记忆类型

**实现方案**:
- 分析现有记忆的类型分布
- 检测哪些类型偏少
- 在空状态页面显示建议卡片
- 新增 `backend/app/services/suggestion_service.py`

**验收标准**:
- [ ] 空状态页面显示类型分布建议
- [ ] 建议基于现有数据动态生成
- [ ] 点击建议快速创建对应类型记忆
- [ ] 已有足够记忆时不显示建议
